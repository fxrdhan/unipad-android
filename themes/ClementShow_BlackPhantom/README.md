# Black Phantom

Converted theme assets from the legacy `BlackPhantom.apk`.

## Source

- APK source reference: <https://youtu.be/BcV9s5QgdU8?si=9sG-tFHWfnWxBFlL>
- Original APK package: `com.kimjisub.launchpad.theme.clementphantomv2`
- Theme name: `Black Phantom`
- Author: `Clement Show`
- Version: `1.0.0`

## Files

- `BlackPhantom_unipad_theme.zip`: ready-to-import ZIP for current UniPad builds
- `theme.json`, `theme_ic.png`, `playbg.png`, `btn.png`, `btn_.png`, `phantom.png`, `phantom_.png`, `chain.png`, `chain_.png`, `chain__.png`: files used by the current ZIP theme format
- `play.png`, `play_.png`, `prev.png`, `prev_.png`, `next.png`, `next_.png`, `pause.png`, `pause_.png`, `appicon.png`: preserved legacy APK assets for reference

## Direct ZIP Download

- ZIP download: [BlackPhantom_unipad_theme.zip](./BlackPhantom_unipad_theme.zip?raw=1)

## Install

1. Download `BlackPhantom_unipad_theme.zip` from the link above.
2. Open UniPad.
3. Go to `Theme -> Add Theme -> Import ZIP file`.
4. Select the downloaded ZIP file.
5. Apply the imported theme from the theme list.
